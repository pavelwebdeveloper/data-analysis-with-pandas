# Project Title

My E-Commerce Data Analysis App

## Instructions for Build and Use

[Software Demo]()

Steps to build and/or run the software:

1. If the website is not available yet through a link then in order to run the software on your computer check if you have Python installed by running "python --version"
2. If you have Python installed then run "python -m pip install Django"
3. Run "pip install pandas" in order to install Pandas
3. Run "git clone https://github.com/pavelwebdeveloper/data-analysis-with-pandas.git" locally on your computer in CMD
4. Download Ecommerce_Sales_Data_2024_2025.csv file from https://www.kaggle.com/datasets/prince7489/e-commerce-sales if it is not included in the project and put it in e_commerce_data folder.
4. Start server by running "python manage.py runserver" inside e_commerce_data_analysis folder
5. In your browser in the address bar type "http://127.0.0.1:8000/"

Instructions for using the software:

1. It displays a few pages. On the main page you can see E-Commerce data view with some form elements to select views and to filter the displayed data in the table with all the E-Commerce data that you can see below the form.
2. You can select other views. 
3. In the Aggregating statistics view you can see some general aggregating statistics for the data and you can select to see some additional aggregating statistics from the dropdown list.
4. In the view called "Technical summary of the data" you can see the technical summary information about the data.
5. In the Plots view you can choose some plots to see about the data.
